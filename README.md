## Installation
```diff
- REQUIRES LINUX!
```

- Clone this repo to your local machine using
`https://github.com/4rlenrey/Learning_SFML`

- install SFML
```shell
$ sudo apt-get install libsfml-dev
```

- Make
```shell
$ make
```

- Run
```shell
$ ./program
```
